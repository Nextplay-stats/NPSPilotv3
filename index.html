Top User Tips
Reports are organised by categories and can be viewed by using the menu at the top of the page. You can only view reports that you have been granted access to. If you cannot find a specific report this means you do not have permission. Please contact Nexplaystats to discuss access requirements.
Please don't forget to use the filters in our reports to show exactly the data you're looking for, helping you to learn and improve.
If you need help using a particular report, get in touch with NextPlay Stats for a refresher: nextplaystats@outlook.com
