Top User Tips
Reports are organised by categories and can be viewed by using the menu at the top of the page.. You can only view reports that you have been granted access to. If you cannot find a specific report this means you do not have permission. Please contact your line manager to discuss access requirements.
Don't forget to use the filters at the tops of each report to show exactly the data you're looking for. This will usually help you find the answers you need.
If you need help using a particular report, get in touch with Business Intelligence for a refresher: datarequests@syfire.org.uk
How to Request a Report Change

If you require further support in using this site, or require a report that you cannot find on NPS, please send an email to our nextplaystats@outlook.com.
